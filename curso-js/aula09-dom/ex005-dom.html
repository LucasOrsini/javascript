Oque é a Árvore DOM?

DOM é a representação em árvore de um documento HTML ou XML. É uma interface de programação que permite acessar e manipular o conteúdo, estrutura e estilo de um documento. A árvore DOM é composta por nós, onde cada nó representa uma parte do documento, como elementos, atributos e texto. Com o DOM, é possível criar, modificar e excluir elementos dinamicamente usando JavaScript.

Maneiras de navegar entre os elementos:

  • Marca/Tag

    • getElementsByTagName()

  • ID

    • getElementById()

  • Nome

    • getElementsByName()

  • Classe

    • getElementsByClassName()

  • Seletor/CSS

    • querySelector() ou querySelectorAll()
